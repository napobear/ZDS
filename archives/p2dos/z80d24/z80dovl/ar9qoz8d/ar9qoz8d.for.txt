AR9QOZ8D is an overlay file for ARUNZ09Q that provides support
of Z80DOS type of date storage for recall with the $D parameter
of ARUNZ09Q. It supports Y,M,D,H,C,N,A requests of the current
system time. AR9QOZ8D.3EX is for .3OM and AR9QOZ8D.HEX is for
the .COM file. Sorry, no support for .4OM file.
