DD is a ZCPR3 disk directory utility with several command line
options, including the option to display only those specified files
which also have a selected combination of the file attributes F1-F7
set.  Can display only non-archived (changed) files as an aid in
file backup.  Operates with GO command.  The display is paged, with
the summary line last.  Great with RAM disks.  Companion to ACOPY,
RENAMZ, ERAZ and UNERAZ.  See DD17.DZC.  (DD17 is a bug fix.)
