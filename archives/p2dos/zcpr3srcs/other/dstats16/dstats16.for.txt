ZCPR3 disk/user statistics utility displays directory and user
area statistics, a list of active user areas with any label
filename found for each, the number of files, directory entries
and disk space used, and summarizes the total number of files,
directory entries and disk free space.  Can also display a disk
allocation bitmap except under Z3PLUS and banked ZSDOS2+.  Version
1.6 displays entire label filename/filetypes.  See DSTATS16.HZP.
